## Installation
1. Clone this repository to your local machine:
```
https://github.com/FGiannini95/BestSellerTest.git
```
2. Move to the application folder:
```
cd BestSellerTest /cd rick
```
3. Install the dependencies:
```
npm i
```
4. Start the application:
```
npm run dev
```
